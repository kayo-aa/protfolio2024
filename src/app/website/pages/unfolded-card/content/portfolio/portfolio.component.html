Project 1

Hardware Heaven: a responsive design

Hardware Heaven is a project where I test my skills with Angular and encourage myself to design. I consume an API to display a list of products with the possibility of adding items to a shopping cart and filtering said list by categories. Additionally, there is a user registry that is saved locally.
The design was carried out by myself and I use Angular Material components for the layout.
